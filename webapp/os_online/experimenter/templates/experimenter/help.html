OpenSesame online is an open source software with which one can create complex experiments using stimuli measurements. The software can then be used to run the experiments. The OpenSesame online project aims to to develop online functionality for OpenSesame, so that is becomes possible to do experiments online using a web browser. Currently, the items that are supported by OpenSesame online are:

  • Text_display
  • Loop
  • Sequence
  • Logger
  • Keyboard_response
  • Sketchpad (in the sketchpad currently only the subitems text tool, image and fixation dot are supported)
  • Reset_feedback
